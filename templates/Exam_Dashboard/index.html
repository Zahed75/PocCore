POC ACADEMY EXAM APP API

This is an Private JSON API for Poc Academy Exam Management System Tools which is associated with the Django Rest Framework And Docker Based (super light-weight Virtual machine which has all necessery programs and dependencies in order to run the program. we setup big application which has UI, database, and lots of microservices we need to do a setup and each of this services need some kind of program) constructed language POC Academy Student Interface, a constructed language POC Academy Admin Interface, a language designed to be as simple and as easy to and Do Exam Easily and Can Arrange Olympiad Exam Easily and many more features.

Endpoint #1: /word/<string:word>
Look up information associated to a word.

This endpoint allows you to find information associated with the word supplied. It returns an JSON list containing a set of objects, each object representing a word that could possibly be linked to the word given.

Example requests
/word/a
 
/word/ijo
 
/word/pakala
 
/word/xyz
